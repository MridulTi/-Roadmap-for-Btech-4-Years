##   A. Setting up the Project

*1.*  Fork [this](https://github.com/Amit366/FunwithPhysics.git) repository.

<p align="center"><img src="https://www.w3schools.com/GIT/img_github_fork.png" width="500" height="200" /></p>\


*2.*  Clone your forked copy of the project.


git clone https://github.com/<your_username>/-Roadmap-for-Btech-4-Years.git

<p align="center"><img width=35% src="https://www.w3schools.com/GIT/img_github_clone_url.png" width="500" height="200" /></p>


*3.* Navigate to the project directory :file_folder: .


cd FunwithPhysics


*4.* Add a reference to the original repository.


git remote add upstream https://github.com/MridulTi/-Roadmap-for-Btech-4-Years.git


*5.* Check the remotes for this repository.


git remote -v


*6.* Always take a pull from the upstream repository to your master branch to keep it at par with the main project(updated repository).


git pull upstream master
